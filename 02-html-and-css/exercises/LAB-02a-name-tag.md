# Name Tag

Recreate a classic name tag in HTML.

1. **Have a plan.** Spend a few minutes planning your layout and styling approach
2. Create semantic HTML
3. Add Layout CSS
4. Add Stylistic CSS

Consider:

- Height and Width
- Margin
- Borders and Border Radius
- Background Color
- Text 
    - Color
    - Text Alignment
    - Text Decoration
- Font Family and Size
